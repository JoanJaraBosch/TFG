{% trans "Monitorització i manteniment" %}

{% blocktrans %} En aquest subapartat, pots decarregar la plana web local per al manteniment i monitorització de les plaques odroid. Aquesta plana et permetrà veure els estats essencials de les plauqes com pot ser la entropia, la calor, el usage de les cpus i altres. A més a més, proporciona un apartat de manteniment on disposaràs de 3 petites terminals on 1 serà per el input, i les altres pels outputs (stout i stderr) on podràs executar petites comandes. A continuació teniu l'enllaç de descàrrega. {% endblocktrans %}
